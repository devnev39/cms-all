# CMS (Canteen Management System)

This is a backend project to be written in spring to for canteen management system.

This will include the following features:

- User management
  - Admin
  - Catering services
  - Catering staff
  - End users
- Canteen Management
  - Items management
  - Payment management
  - Order inspect
- RBAC (Role based access control)
